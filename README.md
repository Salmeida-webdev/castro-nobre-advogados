# Castro Nobre Advocacia — Site Institucional Premium

Site institucional premium desenvolvido para um escritório de advocacia empresarial com foco em posicionamento corporativo, branding jurídico high-ticket e experiência visual sofisticada.

## Visão Geral

O projeto foi desenvolvido com foco em:

- Autoridade institucional
- UX premium
- Branding corporativo
- Design editorial jurídico
- Performance e responsividade
- Conversão institucional
- Aparência de escritório internacional

A proposta visual utiliza:

- Tipografia editorial serifada
- Ouro fosco sofisticado
- Fundo grafite profundo
- Estrutura geométrica minimalista
- Motion design elegante
- Layout cinematográfico
